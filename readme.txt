== Description ==

The Outbound Link Manager monitors outbound links in your posts, easily allowing you to add or remove a nofollow tag, update anchor texts, or remove links altogether.

= Here's what you can do: =

1. Remove links. The original anchor text is kept in tact.
1. Add or remove the NoFollow tag. 
1. Achieve the above in bulk by selecting numerous posts.
1. Edit links. This gives you instant access to the whole link code, allowing you to add styles, targets or change the anchor text.

This plugin is aimed at SEO managers who would like to monitor external links. It is also helpful to monitor external links if you are outsourcing your content creation.

== Installation ==

1. Upload the outbound-link-manager directory (including all files within) to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in WordPress
3. Select Links Manager from your sidebar and start editing!
